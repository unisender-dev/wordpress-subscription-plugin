The plugin allows you to send emails from WordPress to Unisender

== Description ==

The plugin allows you to send emails from WordPress to Unisender.

1.Install the plugin in wordpress.
2 Configure the subscription field. 
3. Connect it to unisender.
All emails of your subscribers will be sent to your account in Unisender.
